# SIGNX AI-Based Sign Language Recognition System
AI-based Sign Language Recognition and Speech Conversion System using MediaPipe, Machine Learning, and Flask for real-time gesture detection and communication. 

## 📌 Overview
SignX2 is an AI-based system that recognizes hand gestures in real-time and converts them into text and speech. It helps bridge the communication gap between sign language users and non-sign language users.

## 🚀 Features
- Real-time hand gesture recognition
- MediaPipe hand landmark detection
- Machine learning-based classification
- Speech output using text-to-speech
- Flask web interface

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- Scikit-learn
- Flask
- pyttsx3
